🗑️ 删除 README 文件 · 打包剩余内容

📌 功能说明
• 选择一个根文件夹，工具会递归扫描所有子文件夹。
• 自动排除文件名包含 "README"（不区分大小写）的文件（例如 README.md, readme.txt, sub/README.pdf）。
• 其余所有文件将被打包成一个 ZIP 压缩包 供您下载。
• 原始文件不会被修改，下载的 ZIP 就是“删除 README 后的完整副本”。
• 可选：将全部文件转换为 .txt 格式（文件名保留原扩展名基础部分）。
等待选择文件夹...
⚠️ 注意：
• 需要浏览器支持 File System Access API（Chrome / Edge 86+）。
• 二进制文件（图片、PDF 等）若转为 .txt 会变成乱码，建议仅对纯文本文件开启转换。
• 打包过程需要读取所有文件内容，文件较多时请耐心等待。
• 下载的 ZIP 内文件为“扁平化”存储（所有文件都在根目录），文件名会包含原相对路径（用下划线连接）以避免重名。
