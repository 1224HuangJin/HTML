📁 递归批量转 TXT · 文件夹路径作为文件名

✨ 功能说明
• 选择一个根文件夹，工具会递归扫描所有层级的子文件夹。
• 每个文件（无论多深）都会被读取并转为 .txt 格式。
• 新文件名 = 相对路径（用下划线连接）_原文件名.txt
  例如：根/父/子/数据.xlsx → 父_子_数据.txt
• 所有新文件打包为一个 ZIP 下载，原文件完全不受影响。
• 适合整理大量文本类文件（代码、日志、CSV、TXT、Markdown 等）。
等待选择文件夹...
⚠️ 注意：
• 需要浏览器支持 File System Access API（Chrome / Edge 86+）。
• 二进制文件（图片、PDF、Word等）转为 .txt 会变成乱码，请仅用于纯文本。
• 递归遍历会包含全部子文件夹，文件数量很多时请耐心等待。
• 最终 ZIP 内的文件路径是扁平的（所有文件都在根目录），文件名已包含原路径信息。
